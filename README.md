# Sistema de Gestión de Estudiantes

Aplicación desarrollada en **Java** como práctica de Programación Orientada a Objetos (POO), utilizando clases, objetos, atributos, constructores y métodos.

---

## Objetivo

Desarrollar una aplicación sencilla que permita gestionar la información de estudiantes mediante los principios básicos de la Programación Orientada a Objetos.

---

## Tecnologías utilizadas

- Java
- IntelliJ IDEA
- Eclipse Temurin JDK 21
- Git
- GitHub

---

```
SistemaGestionEstudiantes
```

---

## Funcionalidades

El sistema permite:

- Registrar estudiantes.
- Mostrar la información de cada estudiante.
- Actualizar el promedio.
- Cambiar la carrera.
- Visualizar los cambios realizados.

---

## Conceptos aplicados

- Clases
- Objetos
- Atributos
- Constructores
- Métodos
- Encapsulamiento
- Instanciación de objetos

---

## Clase principal

La aplicación utiliza la clase `Main` para crear tres objetos de tipo `Estudiante` y demostrar el funcionamiento de los métodos implementados.
